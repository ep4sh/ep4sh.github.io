Subj: DAC на MS SQL Server EXPRESS

По умолчанию в этой редакции(EXPRESS) MS SQL Server режим DAC (Dedicated Admin Connection) отключен. НО! Режим DAC может быть включен насильно:

  1. Включить SQL Server Configguration Manager – > Службы SQL Server
  2. ПКМ -> SQL Server Express-> Свойства, идем в вкладку “Параметры запуска”
  3. В соответсвующее поле добавляем флаг трассировки -T7806
  4. Перезапускаем службежку
  5. СТАРТУЕМ! 
    
              sqlcmd -S .\sqlexpress -Uвашюзер -Pвашпароль -A
